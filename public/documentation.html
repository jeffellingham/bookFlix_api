bookFlix Documentation Page

The bookFlix api will initially be a provider of information about various movies based on novels, comics, manga, or other formats of the written word. It'll include information on the movie, directors, actors, genres, and more. Users will be able to sign up, update their personal information, and create a list of their favorite movies from the site's selection. Our selection will be sparse in the beginning, though we plan to continually add all your favorites!

The dream for the future of bookFlix is for it to become an invaluable resource for users to catalog, rate, and rank every movie (and hopefully TV shows too!) they've watched and be able to use the app to find recommendations on what to watch next.

Business Logic URL HTTP Method Request body format Response body format Example response format

For example: { "genre": { "name": "Action", "description": "Movies in the action genre are defined by risk and stakes. While many movies may feature an action sequence, to be appropriately categorized inside the action genre, the bulk of the content must be action-oriented, including fight scenes, stunts, car chases, and general danger." }, "director": { "name": "John McTiernan", "bio": "John Campbell McTiernan Jr. is an American filmmaker. He is best known for his action films, including Predator, Die Hard, and The Hunt for Red October.", "birth": "1951" }, "_id": "64c1785b2568222783789acb", "title": "Die Hard", "description": "A New York City police officer tries to save his estranged wife and several others taken hostage by terrorists during a Christmas party at the Nakatomi Plaza in Los Angeles.", "imagePath": "diehard.png", "featured": false, "releaseYear": 1988, "actors": [ "Bruce Willis", "Alan Rickman" ] }
Get data on single movie /movies/:title GET None A JSON object holding data about a single movie, contains description, genre, director, image URL, featured or not, release year, and actors. See movie example above.
Get all movies with an actor /movies/actors/:actor GET None A JSON object for every movie attributed to the selected actor, each object is formatted like the example above and includes all the same data. See movie example above.
Get data on genre by name /genre/:genreName GET None A JSON object containing a brief description of the selected genre. For example: { "name": "Action", "description": "Movies in the action genre are defined by risk and stakes. While many movies may feature an action sequence, to be appropriately categorized inside the action genre, the bulk of the content must be action-oriented, including fight scenes, stunts, car chases, and general danger." }
Get all movies of a genre /movies/genre/:genreName GET None A JSON object for every movie attributed with the selected genre, each object is formatted like the movie example above and includes all the same data. See movie example above.
Get data on director by name /directors/:directorName GET None A JSON object containing the director's name, bio, birth year, and death year (if applicable). For example: { "name": "Stanley Kubrick", "bio": "Stanley Kubrick was an American film director, producer, screenwriter and photographer. Widely considered one of the greatest filmmakers of all time, his films—almost all of which are adaptations of novels or short stories—cover a wide range of genres and feature innovative cinematography, dark humor, realistic attention to detail and extensive set designs.", "birth": "1928", "death": "1999" }
Get all movies by a director /movies/directors/:directorName GET None A JSON object for every movie the director is attributed to, each object is formatted like the movie example above and includes all the same data. See movie example above.
Create account for a new user /users POST A JSON object containing the new user's required username, password, and Email, and optionally their birthday as well. For example: { "username": "jimBobAlpha", "password": "squidBillies666", "email": "jimbobo63@gmail.com", "birthday": "1963-12-05" } A JSON object containing the newly registered user's account information, including their id, username, password, email, birthday (if applicable), and favoriteMovies (empty at creation). For example: { "username": "jimBobAlpha", "password": "squidBillies666", "email": "jimbobo63@gmail.com", "birthday": "1963-12-05T00:00:00.000Z", "favoriteMovies": [], "_id": "64c599009020cc6cd0812458", "__v": 0 }
Update user's account info /users/:username PUT A JSON object containing the user's updated account information. Whether it's their username, password, email, birthday, or all the above. For example: { "username": "alphaJimBob", "password": "squidBillyBobThornton666" } A JSON object of the user's updated account information, including their id, username, password, email, birthday, and favoriteMovies. For example: { "_id": "64c599009020cc6cd0812458", "username": "alphaJimBob", "password": "squidBillyBobThornton666", "email": "jimbobo63@gmail.com", "birthday": "1963-12-05T00:00:00.000Z", "favoriteMovies": [], "__v": 0 }
Update user's favorite movie list /users/:username/:movieID POST None A JSON object of the user's updated account information, including their id, username, password, email, birthday, and favoriteMovies. For example: { "_id": "64c5870a78c7b4b0ad9fc0fa", "username": "alphaJimBob", "password": "squidBillyBobThornton666", "email": "jimbobo63@gmail.com", "birthday": "1963-12-05T00:00:00.000Z", "favoriteMovies": [ "64c1785b2568222783789acb", "64c193842568222783789adc" ], "__v": 0 }
Get user's favorites list /users/:username/favorites GET None A JSON object containing ID references of all of the user's favorite movies. For example: [ "64c1785b2568222783789acb", "64c193842568222783789adc", "64c19cec2568222783789ae4", "64c185122568222783789ad0" ]
Delete movie from user's favorite list /users/:username/:movieID DELETE None A JSON object of the user's updated account information, including their id, username, password, email, birthday, and favoriteMovies. For example: { "_id": "64c5870a78c7b4b0ad9fc0fa", "username": "alphaJimBob", "password": "squidBillyBobThornton666", "email": "jimbobo63@gmail.com", "birthday": "1963-12-05T00:00:00.000Z", "favoriteMovies": [ "64c1785b2568222783789acb" ], "__v": 0 }
Delete user account (deregister) /users/:username DELETE None A text message confirming whether the user's account was deleted.
